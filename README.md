# SFC Festival Guide

慶應SFC 学園祭向けのモバイルファーストなデジタルパンフレット / ナビゲーションアプリです。

## 起動方法

```bash
cd "sfc-festival-guide-admin-dropdown-food-stage"
python3 -m http.server 4173
```

Chromeで以下を開きます。

```text
http://localhost:4173
```

管理画面は以下です。

```text
http://localhost:4173/#/admin
```
